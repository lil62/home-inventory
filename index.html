Firebase セットアップガイド
📋 初回セットアップ手順（無料・5分）
  1. Firebase Console にGoogleアカウントでログイン
  2. 「プロジェクトを追加」→ プロジェクト名を入力（例: ouchi-inventory）→ 作成
  3. 左メニュー「構築」→「Realtime Database」→「データベースを作成」
  4. ロケーション選択 → 「テストモードで開始」→ 有効にする
  5. 左メニュー ⚙️ → 「プロジェクトの設定」→「全般」タブ下部の「マイアプリ」→ ウェブアプリ追加（</>アイコン）
  6. 表示された apiKey と databaseURL を上のフォームに入力
🔒 セキュリティルール設定

Realtime Database → 「ルール」タブで以下を設定：

{
  "rules": {
    "$room": {
      ".read": true,
      ".write": true
    }
  }
}

💡 仕組み: 合言葉（共有キー）がルームIDになります。同じ合言葉を入力した端末同士でデータが同期されます。合言葉を知らない人はデータの場所が分かりません。

📊 無料枠: Spark プラン — 保存1GB / 転送10GB/月。家庭用なら十分です。
